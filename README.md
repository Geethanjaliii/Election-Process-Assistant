# Election Process Assistant

A modern Vite + React app that guides first-time voters through a structured election journey.

## Features

- Step-based election flow from registration to results
- Timeline visualization for progress tracking
- Context-aware assistant panel with quick actions
- Responsive layout for desktop and mobile

## Development

Install dependencies and start the app:

```bash
npm install
npm run dev
```

Build for production:

```bash
npm run build
```
